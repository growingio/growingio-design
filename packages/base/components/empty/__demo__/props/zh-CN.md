### Empty

| 参数名      | 描述             | 类型               | 默认值 |
| ----------- | ---------------- | ------------------ | ------ |
| imgSrc      | 将图标替换为图片 | string             | `-`    |
| description | 显示文案         | ReactNode          | `-`    |
| icon        | 自定义显示图案   | ReactNode          | `-`    |
| className   | 节点类名         | string \| string[] | `-`    |
| style       | 节点样式         | CSSProperties      | `-`    |
| size        | 组件尺寸         | 'small' \| 'large' | `-`    |
